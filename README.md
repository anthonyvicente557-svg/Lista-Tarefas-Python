# Lista de Tarefas em Python

Projeto desenvolvido para praticar lógica de programação e manipulação de arquivos em Python.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Remover tarefas
- Editar tarefas
- Marcar tarefa como concluída
- Salvar tarefas em arquivo

## Tecnologias
- Python
- VS Code

## Como executar

1. Clone o repositório
2. Execute o arquivo:

python lista_tarefas.py
